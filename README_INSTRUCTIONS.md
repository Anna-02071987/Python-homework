# Инструкция для наставника 
 
## Установка зависимостей 
1. Установите Python 3.8 или выше 
2. Установите зависимости: `pip install -r requirements.txt` 
 
## Настройка API ключа 
1. Создайте файл `.env` в корневой директории проекта 
2. Добавьте в него ваш API ключ Yougile: 
``` 
YOUGILE_API_KEY=ваш_api_ключ_здесь 
``` 
 
## Запуск тестов 
Для запуска тестов выполните команду: 
```bash 
pytest 08_lesson/ 
``` 
 
## Проверка стиля кода 
Для проверки стиля кода выполните: 
```bash 
flake8 08_lesson 
``` 
 
## Примечания 
- Тесты создают временные проекты и удаляют их после выполнения 
- Для работы тестов необходим действительный API ключ Yougile 
- Все чувствительные данные вынесены в .env файл
