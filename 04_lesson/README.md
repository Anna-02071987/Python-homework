Lesson 4 — Тестирование StringUtils
 Описание

В рамках домашнего задания к уроку 4 были написаны и запущены автоматические тесты для класса StringUtils.

Тестирование выполнено с использованием библиотеки pytest.

Структура задания
04_lesson/ ├── string_utils.py # Реализация класса StringUtils ├── test_string_utils.py # Тесты для класса StringUtils └── README.md 
Описание задания
 
Что тестировалось

Для класса StringUtils покрыты тестами следующие методы:

    capitalize
        обычная строка
        пустая строка
    trim
        строка с пробелами в начале
        пустая строка
    contains
        положительный кейс (символ найден)
        отрицательный кейс (символ не найден)
    delete_symbol
        удаление одного символа
        случай, когда символ отсутствует в строке

Все тесты успешно выполняются.
 
 Как запускались тесты
Тесты запускались командой:

python -m pytest
Используемая версия pytest
pytest 9.0.2
