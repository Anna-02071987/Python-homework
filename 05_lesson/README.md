Lesson 5 — Selenium WebDriver (Basics)

В этом уроке выполнены базовые упражнения по работе с Selenium WebDriver на Python.
Цель — закрепить навыки взаимодействия с веб-элементами и браузером.
Содержание

В папке находятся 4 скрипта:
- lesson05_task1.py

Клик по кнопке с CSS-классом

    Браузер: Google Chrome 
    Страница: http://uitestingplayground.com/classattr
    Поиск кнопки по CSS-селектору
    Клик по синей кнопке

- lesson05_task2.py

Клик по кнопке без ID

    Браузер: Google Chrome 
    Страница: http://uitestingplayground.com/dynamicid
    Поиск кнопки без использования id
    Устойчивый локатор (CSS-селектор)

- lesson05_task3.py

Работа с полем ввода

    Браузер: Mozilla Firefox 
    Страница: http://the-internet.herokuapp.com/inputs
    Ввод текста Sky
    Очистка поля
    Ввод текста Pro
    Корректное закрытие браузера (quit())

- lesson05_task4.py

Форма авторизации

    Браузер: Mozilla Firefox 
    Страница: http://the-internet.herokuapp.com/login
    Ввод логина и пароля
    Нажатие кнопки Login
    Вывод сообщения об успешной авторизации в консоль
    Корректное закрытие браузера (quit())

     Python 3
     Selenium WebDriver
     Google Chrome
     Mozilla Firefox
